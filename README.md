# 📝 Todo List App

Meu primeiro projeto utilizando **React + Tailwind CSS**, desenvolvido com o objetivo de praticar conceitos fundamentais de front-end moderno, manipulação de estado e componentização.

## 🚀 Sobre o projeto

Esta aplicação permite gerenciar tarefas de forma simples e intuitiva, com foco em praticar a construção de interfaces interativas utilizando React.

### Funcionalidades

✅ Adicionar tarefas
✅ Editar tarefas
✅ Excluir tarefas
✅ Adicionar tarefa com tecla Enter
✅ Hover actions (botões aparecem ao passar o mouse)
✅ Interface responsiva e moderna
✅ Estilização com Tailwind CSS

## 🛠️ Tecnologias utilizadas

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Tailwind CSS
* React Icons

## 📚 Conceitos praticados

Neste projeto foram trabalhados conceitos como:

* useState
* Renderização de listas com `.map()`
* Manipulação de eventos (`onClick`, `onChange`, `onKeyDown`)
* Atualização imutável de estado (`map`, `filter`, spread operator)
* Componentização
* Responsividade
* UX/UI básica
